## Conversa

Durante o planejamento colaborativo, os participantes devem poder sugerir opções de interesse para a viagem. Essas sugestões podem incluir atrações turísticas, hospedagens, restaurantes, eventos ou outros locais relevantes para o grupo. As sugestões cadastradas ficarão disponíveis para visualização pelos demais participantes e poderão posteriormente participar de processos de votação e seleção para compor o roteiro final da viagem.

## Critérios de Aceitação

| #   | Critério                                                                                        |
| --- | ----------------------------------------------------------------------------------------------- |
| 1   | O sistema deve permitir que participantes adicionem sugestões à viagem                          |
| 2   | O sistema deve permitir informar o nome da sugestão                                             |
| 3   | O sistema deve permitir selecionar o tipo da sugestão                                           |
| 4   | O sistema deve permitir cadastrar uma descrição opcional                                        |
| 5   | O sistema deve permitir informar a localização da sugestão                                      |
| 6   | O sistema deve associar automaticamente a sugestão ao participante que a cadastrou              |
| 7   | O sistema deve armazenar a data de criação da sugestão                                          |
| 8   | O sistema deve disponibilizar a sugestão para visualização pelos demais participantes da viagem |
| 9   | O sistema deve exibir uma confirmação após o cadastro da sugestão                               |
| 10  | Apenas participantes vinculados à viagem podem criar sugestões                                  |
| 11  | O sistema deve impedir a criação de sugestões em viagens encerradas                             |
| 12  | O sistema deve manter o histórico das sugestões cadastradas                                     |

## Cenários de Teste

| Cenário                               | Dado                                         | Quando                                             | Então                                                       |
| ------------------------------------- | -------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------------- |
| Cadastro de sugestão válido           | O participante está vinculado à viagem       | Informa os dados da sugestão e confirma o cadastro | O sistema registra a sugestão e exibe mensagem de sucesso   |
| Sugestão de atração turística         | O participante deseja sugerir uma atração    | Informa os dados da atração e confirma             | O sistema registra a sugestão                               |
| Sugestão de hospedagem                | O participante deseja sugerir uma hospedagem | Informa os dados e confirma                        | O sistema registra a sugestão                               |
| Sugestão de evento                    | O participante deseja sugerir um evento      | Informa os dados e confirma                        | O sistema registra a sugestão                               |
| Nome da sugestão não informado        | O participante acessa o formulário           | Tenta salvar sem informar o nome                   | O sistema informa que o campo é obrigatório                 |
| Participante sem vínculo com a viagem | Um usuário não participa da viagem           | Tenta cadastrar uma sugestão                       | O sistema impede a operação                                 |
| Viagem encerrada                      | A viagem está marcada como encerrada         | O participante tenta cadastrar uma sugestão        | O sistema informa que não é possível adicionar sugestões    |
| Falha ao salvar sugestão              | O participante informa dados válidos         | Ocorre erro durante o processamento                | O sistema informa que não foi possível registrar a sugestão |

## Observações

1. Uma sugestão pode representar:
   - Atração turística
   - Hospedagem
   - Restaurante
   - Evento
   - Outro local de interesse

2. As sugestões cadastradas poderão ser utilizadas posteriormente em votações realizadas pelo grupo.

3. Todas as sugestões devem ficar visíveis para os participantes da viagem.

4. O autor da sugestão deve ser identificado para fins de rastreabilidade e colaboração.

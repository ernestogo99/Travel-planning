## Conversa

Após criar uma viagem, o Organizador deve poder convidar outros usuários da plataforma para participarem do planejamento. Para facilitar esse processo, o sistema deve disponibilizar uma busca por nome ou identificador do usuário, exibindo apenas usuários cadastrados. Após selecionar um usuário, o Organizador poderá enviar um convite para participação na viagem. Os participantes convidados poderão aceitar ou recusar o convite recebido.

## Critérios de Aceitação

| #   | Critério                                                                            |
| --- | ----------------------------------------------------------------------------------- |
| 1   | O sistema deve disponibilizar uma opção para convidar participantes para a viagem   |
| 2   | Apenas o Organizador da viagem pode enviar convites                                 |
| 3   | O sistema deve disponibilizar um campo de busca de usuários                         |
| 4   | O sistema deve permitir pesquisar usuários por nome ou identificador                |
| 5   | O sistema deve exibir apenas usuários cadastrados na plataforma                     |
| 6   | O sistema deve permitir selecionar um usuário a partir dos resultados da busca      |
| 7   | O sistema deve impedir o envio de convite para participantes já vinculados à viagem |
| 8   | O sistema deve permitir enviar o convite ao usuário selecionado                     |
| 9   | O sistema deve registrar o convite enviado                                          |
| 10  | O sistema deve notificar o usuário sobre o convite recebido                         |
| 11  | O participante deve poder aceitar o convite                                         |
| 12  | O participante deve poder recusar o convite                                         |
| 13  | Após a aceitação, o participante deve ser associado à viagem                        |
| 14  | O sistema deve atualizar automaticamente a lista de participantes da viagem         |
| 15  | O sistema deve exibir mensagem de confirmação após o envio do convite               |

## Cenários de Teste

| Cenário                            | Dado                                           | Quando                                              | Então                                                    |
| ---------------------------------- | ---------------------------------------------- | --------------------------------------------------- | -------------------------------------------------------- |
| Busca de usuário por nome          | O Organizador deseja adicionar um participante | Digita o nome de um usuário cadastrado              | O sistema exibe usuários compatíveis com a busca         |
| Busca de usuário por identificador | O Organizador deseja adicionar um participante | Digita um identificador válido                      | O sistema exibe o usuário correspondente                 |
| Convite enviado com sucesso        | O Organizador seleciona um usuário válido      | Confirma o envio do convite                         | O sistema registra o convite e exibe mensagem de sucesso |
| Participante já vinculado          | O usuário já participa da viagem               | O Organizador tenta convidá-lo novamente            | O sistema impede o envio do convite                      |
| Aceitação do convite               | O participante possui convite pendente         | Aceita o convite                                    | O sistema adiciona o participante à viagem               |
| Recusa do convite                  | O participante possui convite pendente         | Recusa o convite                                    | O sistema mantém a viagem sem o participante             |
| Busca sem resultados               | O Organizador realiza uma pesquisa             | Nenhum usuário corresponde aos critérios informados | O sistema informa que nenhum usuário foi encontrado      |
| Falha ao enviar convite            | O Organizador seleciona um usuário válido      | Ocorre erro durante o envio                         | O sistema informa que não foi possível enviar o convite  |

## Observações

1. Apenas usuários cadastrados na plataforma podem ser convidados para uma viagem.

2. O convite deve permanecer pendente até que o participante aceite ou recuse a solicitação.

3. O Organizador pode convidar múltiplos participantes para uma mesma viagem.

4. Participantes só terão acesso ao planejamento da viagem após aceitarem o convite.

## Cartão

| Elemento | Descrição                                                                            |
| -------- | ------------------------------------------------------------------------------------ |
| Como     | Organizador da viagem                                                                |
| Quero    | Solicitar recomendações personalizadas da IA                                         |
| Para que | O grupo receba sugestões compatíveis com seus interesses e características da viagem |

## Conversa

Durante o planejamento da viagem, o Organizador deve poder solicitar recomendações geradas por Inteligência Artificial. Para gerar as sugestões, a IA deve considerar os interesses identificados dos participantes, as informações da viagem e as interações realizadas durante o planejamento. As recomendações podem incluir atrações turísticas, hospedagens, restaurantes e eventos, servindo como apoio para as decisões do grupo.

## Critérios de Aceitação

| #   | Critério                                                                                          |
| --- | ------------------------------------------------------------------------------------------------- |
| 1   | O sistema deve permitir ao Organizador solicitar recomendações personalizadas                     |
| 2   | A IA deve considerar os interesses identificados dos participantes                                |
| 3   | A IA deve considerar as informações da viagem ao gerar recomendações                              |
| 4   | O sistema deve apresentar uma lista de recomendações geradas pela IA                              |
| 5   | As recomendações podem incluir atrações turísticas                                                |
| 6   | As recomendações podem incluir hospedagens                                                        |
| 7   | As recomendações podem incluir restaurantes                                                       |
| 8   | As recomendações podem incluir eventos                                                            |
| 9   | Cada recomendação deve apresentar informações suficientes para apoiar a decisão dos participantes |
| 10  | O Organizador deve poder visualizar todas as recomendações geradas                                |
| 11  | Os participantes da viagem devem poder visualizar as recomendações                                |
| 12  | O sistema deve registrar a data da geração das recomendações                                      |
| 13  | O sistema deve informar quando não for possível gerar recomendações                               |
| 14  | Apenas usuários vinculados à viagem podem visualizar as recomendações geradas                     |
| 15  | As recomendações não devem ser adicionadas automaticamente ao planejamento da viagem              |

## Cenários de Teste

| Cenário                               | Dado                                                               | Quando                                     | Então                                                                                                |
| ------------------------------------- | ------------------------------------------------------------------ | ------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| Geração de recomendações com sucesso  | A viagem possui informações suficientes                            | O Organizador solicita recomendações       | O sistema apresenta sugestões geradas pela IA                                                        |
| Recomendações de múltiplas categorias | A IA possui informações suficientes para análise                   | O Organizador solicita recomendações       | O sistema apresenta atrações, hospedagens, restaurantes e eventos compatíveis com o perfil da viagem |
| Visualização pelos participantes      | Existem recomendações geradas                                      | Um participante acessa a viagem            | O sistema exibe as recomendações disponíveis                                                         |
| Dados insuficientes                   | A viagem possui poucas informações registradas                     | O Organizador solicita recomendações       | O sistema informa que não foi possível gerar recomendações adequadas                                 |
| Nova solicitação de recomendações     | Já existem recomendações anteriores                                | O Organizador solicita novas recomendações | O sistema gera uma nova lista de sugestões                                                           |
| Falha na geração                      | Ocorre erro durante o processamento da IA                          | O Organizador solicita recomendações       | O sistema informa que não foi possível gerar as recomendações                                        |
| Acesso não autorizado                 | Um usuário sem vínculo com a viagem tenta acessar as recomendações | Solicita visualização                      | O sistema impede o acesso                                                                            |

## Observações

1. As recomendações geradas pela IA têm caráter de apoio à decisão e não substituem as escolhas do grupo.

2. A IA pode utilizar informações como avaliações, sugestões e histórico de interações para personalizar as recomendações.

3. Nenhuma recomendação deve ser adicionada automaticamente ao planejamento da viagem.

4. A funcionalidade deve seguir os princípios de Inteligência Artificial Confiável e Centrada no Ser Humano, mantendo o controle final das decisões com os participantes.

## Cartão

| Elemento | Descrição                                                 |
| -------- | --------------------------------------------------------- |
| Como     | Organizador da viagem                                     |
| Quero    | Gerar automaticamente um roteiro utilizando IA            |
| Para que | Eu obtenha uma proposta inicial de planejamento da viagem |

## Conversa

Após identificar os interesses do grupo e gerar recomendações personalizadas, o sistema deve permitir que o Organizador solicite a criação de um roteiro inteligente utilizando Inteligência Artificial. A IA deve analisar as informações da viagem, os interesses identificados dos participantes e as recomendações disponíveis para propor um planejamento inicial contendo atividades distribuídas ao longo do período da viagem. O roteiro gerado deve servir como apoio à tomada de decisão do grupo e poderá ser posteriormente ajustado pelos participantes.

## Critérios de Aceitação

| #   | Critério                                                                               |
| --- | -------------------------------------------------------------------------------------- |
| 1   | O sistema deve permitir ao Organizador solicitar a geração de um roteiro utilizando IA |
| 2   | A IA deve considerar os interesses identificados dos participantes                     |
| 3   | A IA deve considerar as informações da viagem cadastradas                              |
| 4   | A IA deve considerar as recomendações geradas anteriormente                            |
| 5   | O sistema deve gerar uma proposta de roteiro organizada por período da viagem          |
| 6   | O roteiro deve conter atividades sugeridas pela IA                                     |
| 7   | O sistema deve apresentar o roteiro de forma estruturada e compreensível               |
| 8   | O Organizador deve poder visualizar o roteiro gerado                                   |
| 9   | Os participantes da viagem devem poder visualizar o roteiro gerado                     |
| 10  | O sistema deve registrar a data de geração do roteiro                                  |
| 11  | O sistema deve informar quando não houver dados suficientes para gerar o roteiro       |
| 12  | Apenas usuários vinculados à viagem podem visualizar o roteiro gerado                  |
| 13  | O roteiro gerado não deve alterar automaticamente o planejamento da viagem             |
| 14  | O sistema deve permitir gerar novas versões do roteiro posteriormente                  |

## Cenários de Teste

| Cenário                        | Dado                                                        | Quando                                      | Então                                                  |
| ------------------------------ | ----------------------------------------------------------- | ------------------------------------------- | ------------------------------------------------------ |
| Geração de roteiro com sucesso | Existem informações suficientes sobre a viagem              | O Organizador solicita a geração do roteiro | O sistema apresenta um roteiro gerado pela IA          |
| Visualização do roteiro        | Um roteiro foi gerado                                       | O participante acessa a viagem              | O sistema exibe o roteiro disponível                   |
| Dados insuficientes            | A viagem não possui informações suficientes                 | O Organizador solicita a geração do roteiro | O sistema informa que não foi possível gerar o roteiro |
| Nova geração de roteiro        | Já existe um roteiro gerado                                 | O Organizador solicita nova geração         | O sistema cria uma nova proposta de roteiro            |
| Falha na geração               | Ocorre erro durante o processamento da IA                   | O Organizador solicita a geração            | O sistema informa que não foi possível gerar o roteiro |
| Acesso não autorizado          | Um usuário sem vínculo com a viagem tenta acessar o roteiro | Solicita visualização                       | O sistema impede o acesso                              |

## Observações

1. O roteiro gerado pela IA possui caráter de recomendação e não representa uma decisão definitiva do grupo.

2. A IA deve utilizar os interesses identificados dos participantes para personalizar a proposta apresentada.

3. O roteiro pode incluir atividades, atrações, restaurantes, eventos e outras sugestões relevantes para a viagem.

4. O Organizador e os participantes podem utilizar o roteiro como base para o planejamento colaborativo.

5. A funcionalidade deve seguir os princípios de Inteligência Artificial Confiável e Centrada no Ser Humano, mantendo o controle das decisões com os usuários.

## Conversa

Após a geração de um roteiro inteligente pela IA, os participantes da viagem devem poder visualizar a proposta elaborada. O roteiro deve apresentar de forma organizada as atividades, atrações, hospedagens, restaurantes e eventos sugeridos para cada período da viagem. A visualização permite que os participantes compreendam as recomendações geradas e participem das decisões relacionadas ao planejamento final da viagem.

## Critérios de Aceitação

| #   | Critério                                                                                   |
| --- | ------------------------------------------------------------------------------------------ |
| 1   | O sistema deve permitir aos participantes visualizar o roteiro gerado pela IA              |
| 2   | O sistema deve apresentar as atividades sugeridas no roteiro                               |
| 3   | O sistema deve apresentar as atrações sugeridas no roteiro                                 |
| 4   | O sistema deve apresentar as hospedagens sugeridas no roteiro, quando aplicável            |
| 5   | O sistema deve apresentar os restaurantes e eventos sugeridos no roteiro, quando aplicável |
| 6   | O roteiro deve ser apresentado de forma organizada e compreensível                         |
| 7   | O sistema deve exibir a data de geração do roteiro                                         |
| 8   | O sistema deve indicar a versão mais recente do roteiro gerado                             |
| 9   | Apenas usuários vinculados à viagem podem visualizar o roteiro                             |
| 10  | O sistema deve informar quando ainda não existir um roteiro gerado para a viagem           |

## Cenários de Teste

| Cenário                                  | Dado                                                        | Quando                                           | Então                                           |
| ---------------------------------------- | ----------------------------------------------------------- | ------------------------------------------------ | ----------------------------------------------- |
| Visualização de roteiro disponível       | Existe um roteiro gerado para a viagem                      | O participante acessa a funcionalidade           | O sistema exibe o roteiro completo              |
| Visualização por participante autorizado | O usuário participa da viagem                               | Solicita a visualização do roteiro               | O sistema apresenta o roteiro gerado            |
| Roteiro inexistente                      | Nenhum roteiro foi gerado para a viagem                     | O participante acessa a funcionalidade           | O sistema informa que não há roteiro disponível |
| Visualização da versão atual             | Existem múltiplas versões do roteiro                        | O participante acessa a funcionalidade           | O sistema exibe a versão mais recente           |
| Acesso não autorizado                    | Um usuário sem vínculo com a viagem tenta acessar o roteiro | Solicita visualização                            | O sistema impede o acesso                       |
| Atualização do roteiro                   | Uma nova versão foi gerada pela IA                          | O participante acessa novamente a funcionalidade | O sistema apresenta a versão atualizada         |

## Observações

1. O roteiro gerado pela IA possui caráter de recomendação e não representa uma decisão definitiva do grupo.

2. Todos os participantes da viagem devem poder visualizar o roteiro para contribuir com o planejamento colaborativo.

3. A visualização do roteiro não permite alterações diretas em seu conteúdo.

4. O roteiro apresentado deve refletir a versão mais recente gerada pela IA.

5. A funcionalidade contribui para a transparência do processo de recomendação, permitindo que os participantes acompanhem as sugestões produzidas pela IA.

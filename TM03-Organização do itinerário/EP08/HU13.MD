# HU13 - Visualizar cronograma da viagem

## Cartão

| Elemento | Descrição                                                            |
| -------- | -------------------------------------------------------------------- |
| Como     | Participante da viagem                                               |
| Quero    | Visualizar o cronograma da viagem                                    |
| Para que | Eu possa acompanhar as atividades planejadas para cada dia da viagem |

## Conversa

Os participantes da viagem devem poder consultar o cronograma para visualizar as atividades planejadas ao longo do período da viagem. O cronograma deve apresentar os dias da viagem e as atividades associadas a cada data, permitindo que os usuários acompanhem a programação e acessem os detalhes das atividades quando necessário.

## Critérios de Aceitação

| #   | Critério                                                                                                      |
| --- | ------------------------------------------------------------------------------------------------------------- |
| 1   | O sistema deve disponibilizar uma área de cronograma dentro da viagem                                         |
| 2   | O sistema deve exibir os dias compreendidos entre a data inicial e a data final da viagem                     |
| 3   | O sistema deve permitir selecionar um dia específico da viagem                                                |
| 4   | O sistema deve exibir apenas as atividades associadas ao dia selecionado                                      |
| 5   | O sistema deve exibir o nome de cada atividade cadastrada                                                     |
| 6   | O sistema deve exibir o horário das atividades                                                                |
| 7   | O sistema deve exibir a localização resumida das atividades                                                   |
| 8   | O sistema deve identificar visualmente atividades marcadas como marco da viagem                               |
| 9   | O sistema deve exibir as atividades em ordem cronológica de horário                                           |
| 10  | O sistema deve permitir acessar os detalhes de uma atividade selecionada                                      |
| 11  | O sistema deve atualizar a lista de atividades ao trocar o dia selecionado                                    |
| 12  | O sistema deve exibir uma mensagem apropriada quando não houver atividades cadastradas para o dia selecionado |

## Cenários de Teste

| Cenário                          | Dado                                                        | Quando                        | Então                                                        |
| -------------------------------- | ----------------------------------------------------------- | ----------------------------- | ------------------------------------------------------------ |
| Visualização do cronograma       | O participante acessa uma viagem com atividades cadastradas | Seleciona um dia da viagem    | O sistema exibe as atividades daquele dia                    |
| Troca de dia                     | O participante está visualizando o cronograma               | Seleciona outro dia da viagem | O sistema atualiza as atividades exibidas                    |
| Dia sem atividades               | O participante seleciona um dia sem atividades cadastradas  | O cronograma é carregado      | O sistema informa que não existem atividades para aquele dia |
| Atividade marcada como marco     | Existe uma atividade definida como marco da viagem          | O cronograma é exibido        | O sistema destaca visualmente a atividade                    |
| Acesso aos detalhes da atividade | O participante visualiza uma atividade no cronograma        | Seleciona a atividade         | O sistema exibe seus detalhes                                |
| Ordenação das atividades         | Existem várias atividades no mesmo dia                      | O cronograma é carregado      | O sistema apresenta as atividades em ordem de horário        |

## Observações

1. O cronograma deve apresentar apenas atividades pertencentes à viagem selecionada.

2. As atividades devem ser agrupadas por dia da viagem.

3. Atividades marcadas como marco da viagem devem possuir destaque visual em relação às demais atividades.

4. A visualização do cronograma deve estar disponível para todos os participantes da viagem.

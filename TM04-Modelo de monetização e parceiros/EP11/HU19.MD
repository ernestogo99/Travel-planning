# HU19 - Visualizar anúncios na versão gratuita

## Histórico de Mudanças

| Data       | Versão | Descrição            | Autor                  |
| ---------- | ------ | -------------------- | ---------------------- |
| 21/06/2026 | 1.0    | Criação do documento | ernestodalva@gmail.com |

## Cartão

| Elemento | Descrição                                                                       |
| -------- | ------------------------------------------------------------------------------- |
| Como     | Usuário da versão gratuita                                                      |
| Quero    | Visualizar anúncios durante o uso da plataforma                                 |
| Para que | Eu possa utilizar os recursos da plataforma sem realizar uma assinatura premium |

## Conversa

A plataforma disponibiliza uma versão gratuita para seus usuários. Como forma de sustentação financeira, anúncios podem ser exibidos em determinadas áreas da aplicação. Os usuários da versão gratuita devem conseguir utilizar normalmente as funcionalidades da plataforma enquanto visualizam conteúdos publicitários disponibilizados pelo sistema.

## Critérios de Aceitação

| #   | Critério                                                                                           |
| --- | -------------------------------------------------------------------------------------------------- |
| 1   | O sistema deve exibir anúncios apenas para usuários da versão gratuita                             |
| 2   | Os anúncios devem ser apresentados em áreas específicas da plataforma                              |
| 3   | A exibição dos anúncios não deve impedir a utilização das funcionalidades principais da plataforma |
| 4   | O sistema deve permitir a navegação normal mesmo quando anúncios estiverem sendo exibidos          |
| 5   | Os anúncios devem ser identificados visualmente como conteúdo patrocinado                          |
| 6   | Usuários premium não devem visualizar anúncios                                                     |
| 7   | O sistema deve continuar funcionando normalmente caso um anúncio não possa ser carregado           |
| 8   | O sistema deve atualizar os anúncios conforme configuração da plataforma                           |

## Cenários de Teste

| Cenário                        | Dado                                    | Quando                                | Então                                            |
| ------------------------------ | --------------------------------------- | ------------------------------------- | ------------------------------------------------ |
| Exibição para usuário gratuito | O usuário possui plano gratuito         | Acessa a plataforma                   | O sistema exibe anúncios                         |
| Navegação com anúncios         | O usuário possui plano gratuito         | Utiliza funcionalidades da plataforma | O sistema mantém a navegação normal              |
| Usuário premium                | O usuário possui assinatura premium     | Acessa a plataforma                   | O sistema não exibe anúncios                     |
| Falha no carregamento          | Um anúncio não pode ser carregado       | O usuário acessa a tela               | O sistema continua funcionando normalmente       |
| Identificação do anúncio       | Existe conteúdo publicitário disponível | O anúncio é exibido                   | O sistema identifica o conteúdo como patrocinado |

## Observações

1. Os anúncios fazem parte do modelo de sustentação financeira da plataforma.

2. A presença dos anúncios não deve comprometer a experiência de uso das funcionalidades principais.

3. Usuários que optarem pela assinatura premium não visualizarão anúncios.

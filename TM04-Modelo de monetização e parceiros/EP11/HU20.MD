## Conversa

Durante o planejamento da viagem, a plataforma pode apresentar recomendações patrocinadas de hotéis, pousadas, restaurantes, eventos e outros serviços turísticos. Essas recomendações fazem parte do modelo de parceria da plataforma e devem ser apresentadas de forma transparente aos usuários. O objetivo é aumentar a visibilidade de parceiros comerciais sem comprometer a experiência de planejamento da viagem.

## Critérios de Aceitação

| #   | Critério                                                                                                           |
| --- | ------------------------------------------------------------------------------------------------------------------ |
| 1   | O sistema deve permitir exibir recomendações patrocinadas de serviços turísticos                                   |
| 2   | As recomendações patrocinadas devem ser identificadas visualmente como conteúdo patrocinado                        |
| 3   | O sistema deve apresentar informações básicas do serviço recomendado                                               |
| 4   | O sistema deve permitir que o usuário visualize os detalhes do serviço recomendado                                 |
| 5   | Recomendações patrocinadas podem incluir hotéis, pousadas, restaurantes, atrações e eventos                        |
| 6   | O sistema deve exibir recomendações patrocinadas juntamente com outras recomendações da plataforma                 |
| 7   | A identificação de conteúdo patrocinado deve permanecer visível durante a visualização da recomendação             |
| 8   | O sistema deve permitir que o usuário continue utilizando normalmente as funcionalidades de planejamento da viagem |
| 9   | O sistema deve informar quando não existirem recomendações patrocinadas disponíveis                                |
| 10  | Apenas recomendações aprovadas pela plataforma podem ser exibidas como patrocinadas                                |

## Cenários de Teste

| Cenário                              | Dado                                               | Quando                                      | Então                                                               |
| ------------------------------------ | -------------------------------------------------- | ------------------------------------------- | ------------------------------------------------------------------- |
| Exibição de recomendação patrocinada | Existe um parceiro comercial ativo                 | O usuário acessa as recomendações da viagem | O sistema exibe a recomendação identificada como patrocinada        |
| Visualização de detalhes             | Existe uma recomendação patrocinada disponível     | O usuário seleciona a recomendação          | O sistema exibe seus detalhes                                       |
| Ausência de conteúdo patrocinado     | Não existem recomendações patrocinadas disponíveis | O usuário acessa as recomendações           | O sistema informa a indisponibilidade de recomendações patrocinadas |
| Identificação do conteúdo            | Uma recomendação patrocinada é exibida             | O usuário visualiza a recomendação          | O sistema destaca que se trata de conteúdo patrocinado              |
| Utilização simultânea                | Existem recomendações comuns e patrocinadas        | O usuário acessa a área de recomendações    | O sistema exibe ambas sem prejudicar a navegação                    |

## Observações

1. Recomendações patrocinadas fazem parte do modelo de sustentação financeira da plataforma.

2. O sistema deve identificar claramente conteúdos patrocinados para garantir transparência aos usuários.

3. A exibição de recomendações patrocinadas não substitui outras recomendações disponíveis na plataforma.
